Affine Cipher Solver

Notes:

  • An alphabet with $N$ different letters can be represented by the $N$ non-negative integers less than $N$.
  • An affine cipher takes each plain text letter represented by $x$ to the cipher letter represent by $y\equiv \alpha x +\beta\mod{N}$ where $\alpha,\beta$ are integers and $\alpha$ coprime to $N$.
  • Given two letters of the plain text $x_1, x_2$ and their corresponding encrypted letters in the cipher text $y_1, y_2$ it is possible to 'crack the code' by finding the 'key' $(\alpha,\beta)$ - this solution is shown as the labelled purple point $(\alpha,\beta)$
  • A valid code should not give rise to multiple solutions.
